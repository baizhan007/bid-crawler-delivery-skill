# Example Prompts

下面这些话术应该触发本 Skill：

```text
按上次招投标交付模式做这个新网站。
```

```text
这个标讯网站要符合验收标准，标题正文发布时间不能少，CSV 给我分网站放。
```

```text
每个网站独立项目，源码一个 py 文件，中文注释，最后桌面放完整交付文件夹。
```

```text
检查交付包，去掉 SQL 和不用爬的网站，只保留甲方需要的文件。
```

不应该触发的场景：

```text
帮我写一个普通 requests 教程。
```

```text
分析这个 CSV 的销售额。
```

```text
给我做一个网页 UI。
```
